Generate Invoices

Instructions

This tool will create an individual invoice for every active student in the selected class for the current academic year.

  1. First, ensure the Fee Structure is correctly defined for the class.
  2. Select the class you want to generate invoices for.
  3. Provide a clear title (e.g., "Annual Fees 2024-2025").
  4. Set a due date for the payment.
  5. The system will prevent creating duplicate invoices for the same student and title in the same year.
